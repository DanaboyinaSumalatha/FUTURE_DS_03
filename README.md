# Marketing Funnel & Conversion Performance Analysis

## 📌 Project Overview
This project analyzes e-commerce marketing funnel data to identify:
- Conversion rates
- Funnel drop-offs
- User journey behavior
- Product engagement
- Top-performing categories and brands

The dashboard was built using Power BI and Python for data cleaning and preprocessing.

- Product Views
- Funnel Overview
- User Journey Distribution
- Product Engagement Analysis
